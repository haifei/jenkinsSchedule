Patterns to match against the CORS Origin header against. Some examples:
  • http:*.acme.com:80, https:*.acme.com:443 - will accept requests from any acme.com server over either http or https
  • *.acme.com - will accept requests from any acme.com server
  • http*:foo.acme.com - will accept http or https requests only from acme.com on any port
  • foo.acme.com:8443 - will accept requests in any protocol from foo.acme.com only on port 8443
  • * - will accept requests from any origin.
